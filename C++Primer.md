    + EXIT_FILUERE=-1或其他值
### 1.1.1 编译、运行程序
+ C++程序扩展名
    + cpp(常用)
    + cc
    + cxx(Cmake常用)
    + cp
+ MinGW编译器编译cpp文件:g++ ./main.cpp
    + g++ 为编译器名称
    + ./ 指出要编译的文件所在目录,这里指在根目录
    + main.cpp 指出要编译的文件
Windows环境下会编译出一个名为a.exe的可执行文件
Linux下会编译出一个名为a.out的可执行文件
